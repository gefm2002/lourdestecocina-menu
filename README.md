# Lourdes te Cocina · Menú y pedidos

SPA hecha con Vite + React + TypeScript + Tailwind para menú y carrito con admin local.

## Tipo de proyecto

- Menú digital + carrito con admin local.

## Stack

- Vite + React + TypeScript
- TailwindCSS
- SPA con React Router
- Persistencia en LocalStorage

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Deploy en Netlify

- Build command: `npm run build`
- Publish directory: `dist`
- Redirects: `netlify.toml`

## Datos editables

- `src/data/site.json`
- `src/data/menu.mediadia.json`
- `src/data/menu.nocturno.json`
- Admin en `/admin` para editar en vivo (LocalStorage).
